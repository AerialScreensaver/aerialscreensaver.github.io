💡 Still alpha versions. Near feature complete, but bugs expected. 💡

What's new :
- Fix (hopefully) a crash related to Night Shift usage when opening the menu popover
- Improve performance when opening menu popover (First click still needs to be improved but subsequent ones are much faster than before)
- Add left arrow in fullscreen mode to go back to previous video 
- Add quick fades around password prompt in screensaver 
- More bug fixes.

Known issues/Various gotchas :
- Now playing selection is sometimes not sticking on restart. 
- Automatic downloads are *very* lazy. Use the third icon at the bottom of the popover to open the Video Library and manually download some videos if needed in the meantime. 

After you download the app, move it to your `/Applications` or `~/Applications` folder. Auto updates are provided via Sparkle. 

Important : 
- If things don't work, open settings and look at the checklist on top. You should get buttons to resolve issues if any (register/enable). Report issues about this on discord/github.
- You can start fresh if you encounter issues by deleting `/Users/Shared/Aerial`
