Or install with Homebrew

Prefer the command line? Aerial is on Homebrew. Pick one — you don't need both.

$ brew install aerial

Stable. Installs the latest stable version. Recommended.

$ brew install aerial@beta

Beta. Installs prerelease builds when one is available, latest if not — newest features, bugs expected.

Upgrading from Aerial 3.6.3? Launch the app once after installing — it will install and enable the new Aerial 4 screensaver which is now distributed as an App Extension.

Compatibility

  • macOS 15 (Sequoia) or later
  • Apple Silicon and Intel
  • Single-display and multi-display setups

Updates

Aerial uses Sparkle for background auto-updates.

After installing

  1. Open Aerial.app from /Applications.
  2. Follow instructions to setup how you want Aerial to work, then customize what you want to see by checking the video library, or configure the various settings, all accessible from the icon in your menu bar.

Stuck? Check the FAQ or open an issue on GitHub.

On an older Mac?

Aerial 4.0.12 requires macOS 15 (Sequoia) or later. If you're on an older system, two legacy builds are still available — neither is maintained, and neither will receive new features or fixes.

Aerial Companion 1.5.2

The last build of the previous-generation app, for Macs that can't run macOS 15.

Important: after installing, open Companion's settings and disable auto-updates — otherwise it will prompt you to upgrade to a version your Mac can't run.

Download Aerial Companion 1.5.2

Aerial 3.6.3 (.saver)

The classic standalone screensaver bundle from the original repository, for the oldest macOS versions.

No companion app, no auto-updates — just the .saver installed via System Preferences.
